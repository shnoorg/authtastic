# Contributing

## Dependencies

```bash
## nvm install
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.1/install.sh | bash

## node.js install
nvm install 20
```

## Build

```bash
npm run build

## watch mode
npm run build:watch
```

## Dev

```bash
## spins up a Vite server with a dev app
npm run dev
```

## Publish

```bash
npm publish --access public
```
